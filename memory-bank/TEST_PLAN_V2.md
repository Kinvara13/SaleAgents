# SaleAgents v2 测试计划

更新日期：2026-04-21

## 1. 测试分层

### A. 静态覆盖审计

- 页面路由是否存在
- 页面是否真的调用目标 service / API
- service 与后端路由是否一致
- 历史文档是否与当前代码冲突

### B. 后端接口验证

- 以 `backend-v2` 当前注册路由为准
- Python 版本固定为 `3.11`
- 优先验证漂移点与关键主流程

### C. 前端功能验证

- 以页面走查为主
- 当前仓库未内置浏览器自动化框架
- 页面是否可进入、可加载、核心操作可用、错误态可见

## 2. 运行前提

- 后端使用 `python3.11 -m uvicorn app.main:app --host 127.0.0.1 --port 8000`
- `DATABASE_URL_OVERRIDE=sqlite:///./sale_agents_v2.db`
- 前端可执行 `npm run build`
- 测试用账号：`admin / admin123`
