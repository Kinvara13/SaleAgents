# SaleAgents 功能点开发任务清单

## 概述

本项目为投标智能体系统，需按功能点逐一开发前后端。
**排除模块：demo制作（行6-48）全部暂不做**

---

## 一、投标项目清单（行52-53）

### 1.1 投标项目清单 / 展示上述选择投标项目清单
**功能说明：**
1. 填写应标公司、代理人、电话、邮箱、公司地址、开户银行等信息，提取关键信息
2. 展示项目重点信息：星标项、项目预算、截止投标时间、人工阅读与反馈
3. 相关人员确认后将反馈信息输入（仅做✅确认）

**前端：** 项目列表页，支持查看重点信息
**后端：** 项目管理 API（CRUD + 状态管理）

### 4.4 法定代表人（或非法人单位负责人）身份证明或授权委托书（行57）
**功能说明：** 展示原版文件；识别文件内容，是否有可填写字段，支持预览；支持人工修改

## 五、技术建议书（行4, 82）

### 5.1 技术建议书编写
**功能说明：**
1. 结合客户背景、公司背景把这部分内容给到AI
2. AI在解析生成技术建议过程中，要反向要求负责人员提供证明性材料，即能够证明文字内容的佐证材料
3. 技术建议书的编写需结合【技术打分表】和【技术规范书】的要求

**前端：** 技术建议书编辑器，支持AI生成
**后端：** 建议书生成 API

### 5.2 技术建议书打分
**功能说明：** 根据技术建议书的编写结果与【技术打分表】和【技术规范书】的要求进行预打分。同时支持人工修改后的再次打分

## 九、人工确认节点（8项）

以下为系统中的"人工确认节点"功能（需设计成工作流节点）：

1. **标书要求解读结果确认**（行12）：人工审核AI输出的所有解读内容，确认无误后进入下一环节
2. **功能点拆解&演示逻辑确认**（行19）：人工审核功能清单、优先级、演示流程及时长分配，确认无误后进入页面拆解环节
3. **页面拆解&演示规划确认**（行26）：人工审核页面清单、演示顺序、核心区域标注，确认无误后进入素材准备环节
4. **页面制作需求确认**（行33）：人工审核页面设计方案，确认无误后进入下一环节
5. **页面确认**（行35）：人工审核页面实现效果，给出优化建议，AI继续修改；直至确认无误进入下一环节
6. **素材方案&配文确认**（行40）：人工审核页面设计方案、配文、其他素材要求，确认无误后进入下一环节
7. **录制脚本&指引确认**（行43）：人工审核录制脚本、操作指引，确认无误后执行录制
8. **剪辑方案&输出要求确认**（行48）：人工审核剪辑方案、字幕、输出参数，确认无误后执行剪辑

- 框架：Vue3
